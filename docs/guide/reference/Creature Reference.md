## Evolution

All creatures **evolve** after **600 seconds of continuous assignment** to a building. Evolution **doubles** the creature's boost value.

> [!important] The evolution timer resets if you recall the creature. Assign and leave for maximum benefit.

---

## Boost Types

| Type | Effect |
|---|---|
| Speed | Reduces building tick rate (faster production) |
| Efficiency | Chance to not consume input materials |
| Output | Chance to produce double output |
| Defense | Resistance to [[The Hiss & Corruption|Hiss]] damage |
| Area | Increases building's range/area effects |
| All | Small bonus to all of the above |
